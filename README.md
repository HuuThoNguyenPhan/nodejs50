## 🚀 Chạy dự án

```bash
# Clone dự án
git clone https://github.com/HuuThoNguyenPhan/nodejs50.git
cd nodejs50

# Cài thư viện
npm install

# Tạo file .env từ .env.example
cp .env.example .env

# Khởi động
npm run dev
```
